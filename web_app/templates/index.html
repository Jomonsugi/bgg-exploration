Recommendation Systems An Overview

This app provides two different types of recommendation systems.

Collaborative

This type is used in the section labeled "BGG Members". This approach finds other users with similar preferences (based on ratings) using the matrix factorization method called Alternative Least Squares. An optimized model then predicts what you would rate games you have not rated before, then outputs a filtered list with those rated the most highly. Further, the descriptions of all the games are transformed using natural language processing techniques, and then through nonnegative matrix factorization, games are segmented into categories. The top games in the list are then slightly modified to be diversified across these categories. This approach gives you a currated list of twenty games that you should definitely check out if you have not before! Notice you have control over filtering your list a bit further. Options include: "Best Number of Players" (as voted by BGG members), "Minimum Play Time (minutes)" for those of you who like a longer game, and "Maximum Play Time (minutes)" if you happen to be in a time crunch (or just destest long games).



Content-based

If you are not a BGG member that has actively rated games on the site or you just want to try a different type of recommendation out, there is a section labeled "Another way..." above. This is a simpler method, but still a powerful one. Board Game Geek classifies games as belonging to a variety of "Mechanics" and "Categories". There are currently 51 mechanics and 80 categories used to segment games on the BGG site. Games are classified into multiple mechanics and categories. Using these combinations, a hamming distance matrix is derrived, which in turn can be used to find games that are "closest" to each other. If you type in "Pandemic Legacy Season 1" into this recommendation engine, you will notice your list has a lot of Pandemic variations on it! That is because they share many of the same attributes. However, you can there will be plenty of other games on the list and more unique games might return more interesting results. The same user specified options available to the collaborative recommender are available here as well.
